## ğŸ’» Getting Started

**Prerequisites**: Node.js v16+, npm

1. **Clone & install**

   ```bash
   git clone https://github.com/<user>/trivia-app.git
   cd trivia-app
   npm install
   ```
2. **Start development**

   ```bash
   npm run dev
   ```

   Visit [http://localhost:5173](http://localhost:5173).
3. **Build**

   ```bash
   npm run build
   ```
4. **Preview**

   ```bash
   npm run preview
   ```
5. **Test**

   ```bash
   npm run test
   ```
